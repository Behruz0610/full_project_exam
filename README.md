# Online Shop Exam Project

## Requirements
- Python 3.8+
- PostgreSQL

## Setup
1. PostgreSQL'da `online_shop_exam` database yarating.
2. `config.py` faylida o'zingizning database user va password ni yozing.
3. Terminalda:
```bash
pip install -r requirements.txt
```
4. Loyihani ishga tushiring:
```bash
python main.py
```
